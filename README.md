# FISH8
CHIP-8 interpreter in C/SDL2.

![image](https://github.com/MutantAura/FISH8/assets/44103205/af7e48b6-173a-4a0d-9a56-14b49278aff0)

## Building (Linux/WSL2)
`sudo apt-get install libsdl2-dev pkg-config`
`make`

## Usage:
`/build/fish8 <Path to ROM>`
